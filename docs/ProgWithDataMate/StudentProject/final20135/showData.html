世界中の鳥の”渡り”の距離


これは合計9種の鳥が、渡りの時期に飛ぶ飛行距離を表したデータです。
飛行距離は、これら３つの記事から知ることができます。
●BirdLife INTERNATIONALの「 Migration Marathons: 7 unbelievable bird journeys」
●WWF JAPANの「世界渡り鳥の日：渡りの距離は何が決める？」
●Canon Globalの「渡（わた）り鳥はなぜ渡る？」


また、鳥の上にマウスカーソルを置くと、体重を見ることができます。
体重のデータは以下のサイトから見ることができます。
●Animal Diversity Web ADW「Pygoscelis adeliae（アデリーペンギン）」
●Wikipedia「ハシボソミズナギドリ」
●認定NPO法人バードリサーチ「ホウクロシギ」
●Animal Diversity Web ADW「Anser indicus（インドガン）」
●Encyclopedia of Life「Red Knot（コオバシギ）」
●Animal Diversity Web ADW「Sterna paradisaea（キョクアジサシ）」
●Animal Diversity Web ADW「Cygnus cygnus（オオハクチョウ）」
●Encyclopedia of Life「Limosa lapponica（オオソリハシシギ）」
●Wikipedia「オオジシギ」
