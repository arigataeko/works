マウスの位置でグリッド上の円の大きさ変化

マウスのx座標により、グリッド上に置かれた複数の円の大きさが変わります。

Your browser does not support the canvas tag.

 
【リストExpandMatrix】
int x, y;
int sa = 20; //円の中心の間隔

void setup(){
  size(150,150);
  noFill();
  stroke(200, 0, 0);
}

void draw(){
   background(255);
   for(x=5; x<width;  x=x+sa){
     for(y=5; y<height; y=y+sa){
       ellipse(x, y, mouseX, mouseX);
     }
   }
}
