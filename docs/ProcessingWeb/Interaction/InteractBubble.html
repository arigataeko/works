マウスの移動変化量で大きさ変わる

マウスの移動スピードによって描かれる円の大きさが変わります。ラーニングトレイルの「マウスとのインタラクション」セクションのマウスのスピードと同じです。

Your browser does not support the canvas tag.

 クリックで画面クリア 
【リスト Bubble】
float henka;   //マウスの移動距離

void setup() {
  size(150, 150);
  frameRate(30);    
  background(100);
  fill(255, 200); //透明度を指定
}

void draw() {
   henka = dist(mouseX, mouseY, pmouseX, pmouseY); //変化量
   ellipse(mouseX, mouseY, henka, henka);   
}

void mouseClicked(){
   background(100); // 画面クリア
}
