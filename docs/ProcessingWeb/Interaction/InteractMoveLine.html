面の境界線

マウスのy座標により、色面の境界線が変化します。

黄色の四角の高さは、マウスのy座標mouseYの値です。 
rect(0, 0, wide, mouseY);
青色の四角の高さは、画面の残りの高さでwide-mouseYとなります。 
rect(0, mouseY, wide, wide-mouseY);

Your browser does not support the canvas tag.

 
【リストMoveLine】
int wide=150;  // 画面の大きさ

void setup() {
  size(wide, wide); 
  noStroke();
}

void draw() { 
  fill(255, 220, 0);  //黄色
  rect(0, 0, wide, mouseY);
  fill(0, 0, 160);  //青
  rect(0, mouseY, wide, wide-mouseY);
}
